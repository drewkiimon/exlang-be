## Postgresql

- OrbStack is nice
- Run a simple `docker run --name exlang -e POSTGRES_PASSWORD=mysecretpassword -p 5431:5432 -d postgres`
- Example of this URL is as follows:
  `postgresql://postgres:mysecretpassword@localhost:5431/exlang?schema=public`
